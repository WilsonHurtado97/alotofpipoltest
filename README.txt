Para convertir este código en una API, se podría utilizar un framework de desarrollo como Flask, Django o FastAPI.

Primero, se tendría que importar el framework y configurar una aplicación, luego se crearían rutas para las diferentes funciones que se deseen exponer en la API y 
cada ruta llamaría a su función correspondiente y devolvería una respuesta en formato JSON.

La base de datos mas adecuada para almacenar este tipo de información seria una NoSQL como MongoDB
